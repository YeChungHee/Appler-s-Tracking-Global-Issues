# Appler's Tracking Global Issues

🌍 Global Conflict & Disaster Monitor - Real-time 3D Interactive Dashboard

## Overview
This project provides a comprehensive monitoring system for global conflicts, natural disasters, commodity markets, and US stock market indices.

## Features
- 3D Interactive Globe with conflict zone visualization
- Color-coded threat levels (War/Confrontation/Tension/Caution/Protest)
- Natural disaster tracking (Flood/Earthquake/Drought/Snow/Fire)
- Commodity market analysis
- US Stock Market indices (Dow/S&P500/NASDAQ/Futures)
- Economic impact analysis
